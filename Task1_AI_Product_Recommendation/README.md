# 🛍️ Task 1 - AI Product Recommendation System

## 📌 Overview

This project recommends matching fashion products from a product image using Artificial Intelligence.

The uploaded image is captioned using BLIP, the product category is detected, and matching products are recommended from the fashion dataset.

---

## Features

- Image Caption Generation
- Product Category Detection
- Recommendation Engine
- Dataset Search
- Fashion Pairing

---

## Technologies

- Python
- BLIP
- Transformers
- Pandas
- PyTorch

---

## Workflow

Upload Image

↓

Recommend Products

↓

Display Results
