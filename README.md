# 👗 AI Fashion Search and Recommendation System

An end-to-end Generative AI project for fashion product understanding using BLIP and Sentence Transformers.

## 📌 Project Modules

### 🛍️ Task 1 – AI Product Recommendation System
- Generates captions from fashion images using BLIP.
- Detects product categories.
- Recommends matching fashion items.

### 📦 Task 2 – Unique Product Catalog Creation
- Generates embeddings for products.
- Detects duplicate or similar products.
- Builds a unique product catalog.

### 🔍 Task 3 – AI Reverse Product Search Engine
- Finds visually and semantically similar products.
- Uses Sentence Transformers for semantic search.
- Displays the most relevant products from the dataset.

## 🛠️ Technologies Used
